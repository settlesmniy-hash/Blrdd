# PulseTrade Design System

## Color Palette
- `bg-primary`: #0a0d14 — deep near-black navy (app background)
- `bg-card`: #111827 — dark navy card background
- `bg-card-inner`: #1a2235 — slightly lighter inner card / row bg
- `border-card`: #1e2d45 — subtle blue-tinted border
- `accent-purple`: #6366f1 — purple accent (confidence badges, active buttons, Save & Connect)
- `accent-blue`: #3b82f6 — blue (WAIT button border/text)
- `accent-green`: #22c55e — green (positive % values, YES prices)
- `accent-red`: #ef4444 — red (negative % values, BUY recommendation buttons)
- `accent-orange`: #f97316 — orange (toggle pill on Live Crypto)
- `text-primary`: #ffffff — white primary text
- `text-secondary`: #6b7280 — gray secondary/label text
- `text-muted`: #374151 — very muted text
- `live-dot`: #ef4444 — red live indicator dot

## Typography
- Font: System default (SF Pro on iOS)
- Logo: Bold 28px white "PulseTrade"
- Screen titles: Bold 20px white
- Section headings: Bold 18px white
- Card titles: Medium 14px white
- Labels: Regular 11-12px gray
- Percentages: Bold 16-20px (colored)

## Components
- Cards: borderRadius 12, bg #111827, border 1px #1e2d45, padding 14-16
- BUY buttons: bg #ef4444, borderRadius 6, white bold text 11px uppercase
- WAIT buttons: border 1px #3b82f6, borderRadius 6, blue text 11px
- Confidence badge: bg #1e2d45, borderRadius 8, purple bold text
- Live dot: 6px red circle
- Section toggle pill: rounded bg (green=#22c55e, orange=#f97316)
- Stats row: 3 equal columns inside card

## Navigation
- Tab bar: hidden, using Stack navigation with back arrows
- Top header: "PulseTrade" logo + subtitle + time + icon buttons
- Back navigation: "← Back" in blue, centered screen title
